# CyberShield Phishing Detector

## Overview

CyberShield is a web application designed to help users identify potential phishing emails, messages, and suspicious links. The application analyzes user input for common phishing indicators and provides a risk assessment along with security recommendations.

This project was developed as part of the CSE 310 Applied Programming course at BYU-Idaho.

---

## Purpose

The purpose of this project is to demonstrate the use of HTML, CSS, and JavaScript to build an interactive cybersecurity tool. The application helps users recognize common phishing techniques and promotes safer online behavior.

---

## Features

* Detects suspicious phishing keywords and phrases
* Detects unsecured HTTP links
* Detects shortened URLs
* Identifies requests for sensitive information
* Assigns a phishing risk level (Low, Medium, or High)
* Provides security recommendations based on the analysis
* Allows users to clear results and analyze additional messages

---

## Technologies Used

* HTML5
* CSS3
* JavaScript

---

## How to Run the Application

1. Clone or download this repository.
2. Open the project folder.
3. Open `index.html` in a web browser.
4. Paste a suspicious email, message, or URL into the text area.
5. Click **Analyze Message**.
6. Review the risk level, warning signs, and recommendations.

---

## Example Input

```text
URGENT! Your bank account has been suspended. Click here to verify your password immediately: http://fake-bank-login.com
```
