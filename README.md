# CyberShield Phishing Detector

## Overview

CyberShield is a cybersecurity web application that helps users identify suspicious phishing emails, messages, and unsafe links. The application analyzes user input for common phishing indicators and provides a risk assessment.

This project was created for the CSE 310 Applied Programming course.

---

## Features

- Detects suspicious phishing keywords
- Detects unsafe HTTP links
- Detects shortened URLs
- Assigns phishing risk levels
- Provides cybersecurity recommendations
- Allows users to clear and analyze new results

---

## Technologies Used

- HTML
- CSS
- JavaScript

---

## How To Run

1. Download or clone the repository
2. Open the project folder
3. Open `index.html` in a browser
4. Paste a suspicious message or URL
5. Click "Analyze Message"

---

## Example Phishing Message

```text
URGENT! Your bank account has been suspended. Click here to verify your password: http://fake-bank-login.com
```
